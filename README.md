# 🧠 nanomind - Private AI for modest hardware devices

[![](https://img.shields.io/badge/Download-nanomind-blue.svg)](https://github.com/geovanna8297/nanomind/releases)

nanomind brings artificial intelligence to your personal computer. You run this software on your own hardware without sending data to the cloud. It works on systems with limited memory and keeps your information private.

## 📥 How to download the software

Follow these steps to get the application on your computer.

1. Visit [this page to download](https://github.com/geovanna8297/nanomind/releases).
2. Look for the Assets section at the bottom of the newest version.
3. Click the file ending in `.exe` to save it to your Windows computer.
4. Open the file once the download finishes to begin installation.

## ⚙️ System requirements

This software works on most standard Windows computers. Your system needs these items to run the program well:

* Windows 10 or Windows 11.
* A processor with at least 2 gigahertz speed.
* 2 gigabytes of available system memory.
* 500 megabytes of free space on your hard drive.

The software uses the llama.cpp engine to process data. This keeps the application small and fast. You do not need a special graphics card to run these models.

## 🚀 Setting up the application

After you install the program, you must set an initial configuration.

1. Find the nanomind icon on your desktop or in your start menu.
2. Double-click the icon to launch the window.
3. The software checks your hardware settings during the first launch.
4. Select a model from the list. The software uses small, efficient language models by default.
5. Click Save to keep your settings.

## 💬 Using the chat window

The main screen shows a text box at the bottom. Type your question or task in this box and press Enter.

The software generates text based on your prompt. Because the system works offline, it does not send your input to any server. Your data stays on your local machine.

## 🛠️ Performance tips

You can adjust settings to improve performance. Open the Settings menu to change these options:

* Model size: Choose smaller models if the application feels slow.
* Thread count: The default setting matches your processor cores. Change this only if you experience lag.
* Cache limit: Set this to 1 gigabyte to ensure the program leaves room for other Windows tasks.

## 🔒 Understanding privacy

nanomind functions completely offline. The program does not have an internet connection requirement for basic tasks. You can test this by disconnecting your Wi-Fi after the download finishes. All processing happens on your local processor.

## ❓ Frequently asked questions

**Does this software cost money?**
No, nanomind remains free and open-source.

**Can I run this on a Raspberry Pi?**
Yes, the version for Raspberry Pi is available on the release page. Use the file ending in `.zip` for that device.

**Why does the text take time to load?**
The software generates text one word at a time based on pure math. Older processors or systems with exactly 2 gigabytes of memory may take longer to show each response.

**Is my data shared with other companies?**
No. The application lacks trackers or data collection tools. Nothing leaves your laptop.
